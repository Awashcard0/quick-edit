How to

If this is your first time you may want to look at how to use quick edit notebook.

First start by clicking the new button. This will create a new file and save it to your computer. (If you care and it's just a JSON file )

Second click on choose file and select the file that was just downloaded. This will load the file.

Third click new page and name it something you will know
